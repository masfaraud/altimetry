srtm.py
=======
Fast and lightweight querying of SRTM elevation data within Python.

## Getting started
Working with **srtm.py** is simple.
```python
from srtm import ElevationService
e = ElevationService()
elevation = e.get_elevation(50.8, 7.5)
```
